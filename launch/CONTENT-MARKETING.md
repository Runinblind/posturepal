## 🔍 SEO Technical Checklist

**On-Page SEO:**
- [ ] Target keyword in title tag
- [ ] Target keyword in H1
- [ ] Keyword in first 100 words
- [ ] Keyword in URL slug
- [ ] Meta description (150-160 chars)
- [ ] Alt text for all images
- [ ] Internal links (3-5 per post)
- [ ] External links to authority sites
- [ ] Schema markup (Article, HowTo, FAQ)

**Content Quality:**
- [ ] Original research/insights
- [ ] 1,500+ words (for target posts)
- [ ] Scannable (headers, bullets, short paragraphs)
- [ ] Multimedia (images, videos, infographics)
- [ ] Mobile-friendly
- [ ] Fast loading time

**Link Building:**
- Guest posts (backlinks)
- Resource page outreach
- Broken link building
- HARO (Help a Reporter Out) - provide expert quotes
- Infographic outreach (offer embeds)
